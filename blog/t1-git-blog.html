Arrays and hashes

Arrays are lists of single items, like a shopping list, while hashes are dictionaries, where each item is tied to another, like words and their definitions. They're useful because you can look things up in very different ways. In an array, you just look things up by where they appear in the list, first, fifth, etc. But in a hash, you can look up, say, a definition by its word, or an author by a book title.

Positioning in CSS

Today I learned a lot about positioning in CSS, which is still one of the hard things for me here at DBC. The gist is that the HTML by default puts elements in static position. If you want to move them around relative to that ordinary position, you use relative positioning. Fixed position fixes an element in the browser window, so it doesn't move even when you scroll. Helpful for a navbar or something. And absolute positioning positions the element relative to the HTML file, or whatever other element it's inside. I also learned a neat trick for centering absolutely positioned elements, because you can't do it the normal way, with align. Well, that's all for today. Happy blogging!
